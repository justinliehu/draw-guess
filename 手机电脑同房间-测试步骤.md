# 手机和电脑同房间测试步骤

## 1. 在电脑上启动服务器

双击运行 **start-server.bat**。

看到类似输出：
```
  Server running:
  http://localhost:8080
  Phone + PC same room -> use this on BOTH devices:
  http://192.168.1.100:8080/index-standalone.html
```
记下第二段里的 **http://192.168.x.x:8080/...** 地址（你的电脑 IP 可能不同）。

---

## 2. 电脑和手机都用「同一地址」打开

- **电脑**：在浏览器里打开上面显示的地址，例如  
  `http://192.168.1.100:8080/index-standalone.html`  
  **不要用 localhost**，否则手机无法和电脑共用房间。

- **手机**：连上**和电脑同一个 WiFi**，在手机浏览器里输入**同一个地址**：  
  `http://192.168.1.100:8080/index-standalone.html`  
  （把 192.168.1.100 换成你电脑上显示的 IP。）

---

## 3. 防火墙（手机打不开时）

若手机浏览器提示无法打开或一直转圈：

- 在电脑上打开 **Windows 防火墙** → **高级设置** → **入站规则** → **新建规则**。
- 选「端口」→ 下一步 → TCP，特定本地端口填 **8080** → 允许连接 → 下一步 → 全选 → 名称填「游戏 8080」→ 完成。

或临时关闭防火墙再试（仅测试用）。

---

## 4. 测试流程

1. **电脑**：输入昵称 → 进入游戏 → 选「你画我猜」→ 会生成一个房间号（如 1234）。
2. **手机**：输入另一个昵称 → 进入游戏 → 在「已有房间」里点该房间的「快速进入」，或点「输入房间号加入」输入 1234。
3. 两边在「玩家列表」里应能看到对方，并可点「准备」、等倒计时开始游戏。

---

**要点**：电脑和手机都必须用 **同一个 URL**（显示在 start-server.bat 窗口里的那个），这样才会连到同一台服务器、进同一个房间。
